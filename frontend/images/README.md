# Images Folder

Place all image assets here:
- Meal photos uploaded by vendors
- Logo images
- Placeholder images
- Icons

## Recommended Structure:
- `meals/` - Meal photos
- `logos/` - Logo files
- `icons/` - Icon files

## Placeholder Image
If you don't have images yet, you can use a placeholder service:
- https://via.placeholder.com/400x300
- Or create a simple placeholder.jpg file
